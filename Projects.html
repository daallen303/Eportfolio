This pizza website turned out to be a very interesting project. To order a sub online the same way you order a pizza would either be innefficient or take away from order customization. The solution was to have two standard templates one for pizza and calzones, and one for Stromboli, Rolls, and salads, and a unique UI for the subs. All menu items are stored in a database which allows the admin to dynamically add, edit, and delete menu items without breaking the code, using easy to fill forms with validation.

We wanted to create a task manager that was unique and we attempted to accomplish that in a few different ways. For one each task and project has a status assigned to it; new, started, or completed. This allows the user to see what needs to be worked on in an effecient and well organized manner. Also assigned to a project/task is a priority, high, medium, or low. The different colors of the task help the user identify the priority as well as where they are shown on the page (high towards the top, low towards the bottom).
